# Add Reverse String Utility

## Phase 1: Add reverse function

Create a `reverse` utility function in `src/reverse.ts` that reverses a string.

Requirements:
- `reverse(str)` returns the string with characters in reverse order
- Handle empty string input (return empty string)

Add tests in `src/reverse.test.ts`.

### Acceptance Criteria

1. `src/reverse.ts` exports a `reverse(input: string): string` function
2. `src/reverse.test.ts` contains tests for normal strings and empty string
3. All existing tests continue to pass
4. TypeScript compilation passes with no errors
